# XTMF2.Web

Server-side Blazor based GUI for XTMF2.

# Building XTMF2.Web

## Requirements

1. DotNet Core 3.1 SDK

## Clone the XTMF2.Web repository

```
git clone https://github.com/TravelModellingGroup/XTMF2.Web.git
```

## Update Submodules

```
git submodule update --recursive --remote
```

## Building

```
dotnet build 
```
